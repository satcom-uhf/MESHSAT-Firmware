# MESHSAT Firmware

Репозиторий основан на версии https://github.com/meshtastic/firmware от 2022 года. 
Внесены мелкие правки для работы на геостационарах. 
В экспериментах принимали участие:
- Оберон
- Ямал
- Монах
- Кама

Приветствуются пулл-реквесты от желающих продолжать эксперименты.
